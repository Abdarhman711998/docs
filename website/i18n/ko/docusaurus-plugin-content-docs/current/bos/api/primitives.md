Near Social VM은 구성 요소에서 사용할 수 있는 사용자 지정 API 집합을 제공합니다.

표준 객체:
- `JSON` - JSON API입니다.
- `Object` - Object API입니다.
- `Date`
- `console` - 콘솔 API입니다.
- `Math`
- `Array`
- `Number`
- `Buffer`
- `Audio`
- `Image`
- `File`
- `Blob`
- `FileReader`
- `URL`
- `Uint8Array`
- `Map`
- `Set`

가져온 객체:
- `Big` - `big.js` 라이브러리에서 가져온 big number API입니다.
- `BN` - `bn.js` 라이브러리에서 가져온 big number API입니다.
